# Kitap Arşivi Projesi

hocam selamlar, bu proje bizim node.js ödevi için hazırladığım kitap kayıt uygulamasıdır. 
expressjs ve sqlite kullanarak yaptım. mvc yapısına uygun ayırmaya çalıştım.

### projenin özellikleri:
- kitap ekleme, silme ve güncelleme yapılıyor.
- basım yılı ve sayfa sayısı gibi alanlar ekledim.
- ana sayfada toplam kaç kitap var ve toplam kaç sayfa okundu istatistikleri görünüyor.
- render üzerinden canlıya aldım linki aşağıda.

**canlı yayın linki:** https://kitap-arsivim.onrender.com
